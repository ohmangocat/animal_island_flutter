# Third Party Notices

This package bundles visual and font assets so the Flutter components can render consistently across supported platforms.

## React Reference Assets

Some SVG, PNG, and WebP assets are mirrored from the `animal-island-ui` reference project used to guide this Flutter implementation.

- Source project: https://github.com/guokaigdg/animal-island-ui

## Bundled Fonts

The bundled `.woff2` files are extracted from the following `@fontsource` npm packages, using the versions declared by the React reference project.

- `@fontsource/nunito` 5.2.7
- `@fontsource/noto-sans-sc` 5.2.9
- `@fontsource/zen-maru-gothic` 5.2.8

These font packages declare the SIL Open Font License 1.1 (`OFL-1.1`). See the upstream font projects and `@fontsource` package metadata for complete license text and attribution.
